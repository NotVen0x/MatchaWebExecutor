Compatible with Chrome, Kiwi, and Safari.

💻 PC Installation

1. Extract the .zip file.
2. Open chrome://extensions.
3. Enable Developer Mode.
4. Click "Load Unpacked" and select the folder.

📱 Android (Kiwi)

1. Open Kiwi Browser.
2. Go to Extensions > Enable Developer Mode.
3. Tap "+ (from .zip)" and select the Matcha file.

🍎 iOS (Safari)

1. Install the "Userscripts" app from App Store.
2. Enable in Safari Settings.
3. Save Matcha.user.js to your Userscripts folder.
