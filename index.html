DOWNLOAD PC / ANDROID (.ZIP) GET IOS SCRIPT HUB (.JS)

TIP: iOS users must have the "Userscripts" app installed from the App Store.

💻 PC (Chrome/Brave)

1. Extract the .zip.
2. Go to chrome://extensions.
3. Enable Developer Mode.
4. Load Unpacked.

📱 Android (Kiwi)

1. Open Kiwi Browser.
2. Tap 3-dots > Extensions.
3. Enable Developer Mode.
4. Tap + (from .zip).

🍎 iOS (Safari)

1. Download "Userscripts" from App Store.
2. Enable in Safari Settings.
3. Save the Matcha.user.js file into your Safari Userscripts folder.
