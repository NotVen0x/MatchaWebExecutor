Universal support for PC, Android, and iOS (Safari).

💻 PC (Chrome / Brave / Edge)

1
Download and extract the Matcha .zip folder.
2
Open chrome://extensions in your browser.
3
Enable Developer Mode in the top right corner.
4
Click Load Unpacked and select the extracted folder.

📱 Android (Kiwi Browser)

1
Install Kiwi Browser from the Google Play Store.
2
Download the Matcha .zip to your device.
3
Open Extensions in Kiwi and enable Developer Mode.
4
Tap + (from .zip) and select the Matcha file.

🍎 iOS (iPhone / iPad)

1
Install the "Userscripts" extension from the App Store.
2
Enable the extension in Settings > Safari > Extensions.
3
Create a folder in your Files app and set it as the Userscripts directory.
4
Save our Matcha.user.js file into that folder.
